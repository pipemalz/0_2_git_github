Curso profesional de Git y GitHub

Resumen clase 1

  • git init Inicia el repositorio.

Resumen clase 2

  • git log Muestra el historial de commits.

Resumen clase 3

  • git reset --soft Todo vuelve a la version del commit especificado pero queda en el área de staging lo que se tenía antes del reset.

Resumen clase 4

  • git branch Muestra las ramas que existen en el repositorio.

Resumen clase 5

  • git remote -v Muestra los repositorios remotos.

Resumen clase 6

  • git push origin master Envía los cambios del repositorio local de la rama master al repositorio remoto.

Resumen clase 7

  • git remote add (nombre) (url) Crea un nuevo origen de datos para hacer pull y push a un repositorio remoto.
