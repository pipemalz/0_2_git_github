Curso profesional de Git y GitHub

Resumen clase 1

  • git init Inicia el repositorio

Resumen clase 2

  • git log Muestra el historial de commits

Resumen clase 3

  • git reset --soft Todo vuelve a la version del commit especificado pero queda en el area de staging lo que se tenia antes del reset.
